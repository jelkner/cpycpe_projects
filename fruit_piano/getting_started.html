Getting Started

  • The first thing to do is to make sure your device is connected and you are in the "code.py" file. Then import from the Adafruit cpx library, by using the following lines of code: 
             from adafruit_circuitplayground.express import cpx
  • The second step is to set up the slide switch on your Circuit Playground Express. You want to set up the slide switch in order to turn the piano on and off. By switching it left and right. Use the following code to set up your slide switch: 
                while True:
                if cpx.switch:
                    cpx.pixels.fill((0, 0, 0))
                    cpx.stop_tone()
                    continue
