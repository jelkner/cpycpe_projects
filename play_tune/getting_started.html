Getting Started

  1. Be sure to do from adafruit_circuitplayground.express import cpx on the very first line.
  2. Below I have posted some muscial notes that will be required to play the theme song : 
                c = 262
            d = 294
            e = 330
            f = 349
            g = 392
            a = 440
            b = 294
            c2 = 523
            d2 = 587
            e2 = 659
            f2 = 698
  3. Below I have posted the wait time needed for the theme song : 
                quarter = 1.0
            three_quarter = 0.75
            one = 1.0
            two = 2.0
            two_half = 2.5
            three = 3.0
  4. Also create a variable called tempo. This variable will determine how long each note will play.
