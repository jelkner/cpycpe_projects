Boolean Values

To be able to represent the individual bits of our number using NeoPixels, we need to be able to determine the value (1 or 0) of each bit.

The bitwise & operator does this nicely for us. To understand how, start a Python shell (a.k.a. REPL) and experiment with each of the following until you feel you understand what is happening:

  1. Powers of two have only a single non-zero bit in binary (revisit Binary Basics if you aren't nodding your head in aware agreement). The exponentiation operator, **, can be used to generate these values:

      >>> 2 ** 0
  >>> 2 ** 1
  >>> 2 ** 3
  >>> 2 ** 6
  2. >>> bool(0)
  3. >>> bool(1)
  4. >>> bool(2)
  5. >>> bool(6)
  6. >>> bool(99)
