Circuit Playground Express

Display a Number in Binary with Neopixels

By: Jeffrey Elkner

Introduction Binary Basics Python Bit Operators Booleans for Bits Final Product

What You Will Do

You will write a Python program that displays a number in binary on a Circuit Playground Express using the 10 NeoPixels to represent bits.

What You Will Learn

In this activity you will...

  • learn what a binary number is and how to represent decimals in binary.
  • be introduced to Python's bitwise operators.
  • learn about the boolean data type and how Python handles booean values.
  • be introduced to list comprehensions in Python.
  • be introduced to Python's ternary operator.
  • learn how to light NeoPixels on the Circuit Playground Express.
