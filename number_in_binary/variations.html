Python's Conditional Expression

Python has a conditional expression that evaluates to one of two values depending on a Boolean condition. 

>>> condition = True
>>> result = 'Yes!' if condition else 'No!'

The variable result will be set to the value 'Yes!' in this example, since the condition variable is True (otherwise it would be 'No!'.

We can use this with a number to print '1's and '0's for its bits. 

>>> NUMBER = 20 
>>> '1' if NUMBER & 2 ** 0 else '0'
0
>>> '1' if NUMBER & 2 ** 1 else '0'
0
>>> '1' if NUMBER & 2 ** 2 else '0'
1
>>> '1' if NUMBER & 2 ** 3 else '0'
0
>>> '1' if NUMBER & 2 ** 4 else '0'
1

Let's use this to write a little program that prints out the binary digits of a number as 1s and 0s. Since our Circuit Playground Express has 10 NeoPixels, we will change the number of bits to 10. 

NUMBER = 937
bits = '' 

for i in range(10):
    bits = ('1' if NUMBER & 2 ** i else '0') + bits

print(bits)

Running this program will print out the binary representation of 937, which is 1110101001.

List Comprehensions

List comprehensions provide a way to build lists from existing sequences of values inspired by the set-builder notation in mathematics. 

>>> [x ** 2 for x in range(10)]
[0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

We can use a list comprehension to elegantly create our list of Boolean bit values. 

>>> [bool(937 & 2 ** i) for i in range(10)]
[True, False, False, True, False, True, False, True, True, True]

This has the bit in the reverse order for representing the binary number in text, but is just what we want for writing it out to NeoPixels.
