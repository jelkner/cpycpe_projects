GPS Logger

By: Dylan Herreda

Introduction Materials Required boot.py datalogger.py Instructions

What You Will Do

You will create a GPS logger with circuitpython that is able to collect location data that can be used for variet of things.

What You Will Learn

  • You will learn how to create a GPS logger.
  • You can also learn about UART and filesystems if you'd like to expand your understanding of how this project works.
