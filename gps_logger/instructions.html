GPS Logger

By: Dylan Herreda

Introduction Materials Required boot.py datalogger.py Instructions

Instructions

  • Assemble materials:
    • Put both the feather and the GPS on the feather doubler
    • Attach the antenna to the GPS FeatherWing
  • Put code (datalogger.py) on the feather
  • Boot the feather in write mode (only applicable if you aren't using an adalogger)
    • Use the code in boot.py to do it, and connect pin to the serial console to verify that data is being collected.
  • Get data
    • If you are using an adalogger :
      • Connect to a serial console
      • Make the script stop running with CTRL + C
      • Exit the serial console
      • Power down the board by ejecting it and disconnecting the power supply
      • Plug the SD card into your computer and copy the file across
    • If you are using built in storage :
      • Connect to a serial console
      • Stop the script with CTRL + C
      • Exit the serial console
      • Copy the file from the board to your computer using terminal or a graphical file manager
