What You Will Do

You will learn to make the game Mastermind on the Circuit Playground Express.

How to play

The game master mind is a two player game of making and cracking codes. There is a code maker that creates a color code and a code cracker which is supposed to guess the color code.

The code maker creates the code by using the two main buttons, the left one changes the color you select and the right one is to select the color. There's a third attached button that is a submit button for the code

Rules:

  • The code maker goes first, they decide what the secret code will be, and they cannot show the code breaker
  • The code breaker then tries to guess the code the code maker submitted and once they guess they guess the code they win! Each time there's a guess though there are lights that give you hints on what's right or wrong
  • A red light means that color is nowhere in the code
  • A blue light means there's that color in the code but not in that spot
  • A green light means the color is in the code in that spot
