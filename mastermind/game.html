The game running

The last while loop is the actual game loop. It gets a master code and then repeatedly gets a test code, scores it against the master, and then displays it. It stops when the codes match. 

    while True:
        master_code = choose_code()
        while True:
            test_code = choose_code()
            scored = score_code(test_code, master_code)
            if scored == [2, 2, 2, 2]:
                break
            time.sleep(0.2)
            display_score(scored, led)
            time.sleep(0.5)
