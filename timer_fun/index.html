What You Will Do

You will learn to make a timer on the Circuit Playground Express. Every second that goes by, a neopixel will turn red and every 10 seconds you will hear a small sound.

What You Will Learn

  • You learn how to use python libraries to light up neopixel
  • Learn to use the time module
  • Learn to use the sound module

To start things off, start by clicking here. It will take you to the adafruit website where you can make new projects, by pressing New Project.
