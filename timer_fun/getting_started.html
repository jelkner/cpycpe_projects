Getting Started

  1. Before we get to the real code, we have to import libraries from Adafruit. Write down the following lines of code. 
      import time
  from adafruit circuitplayground.express import cpx
  2. Start by making a variable counter.This variable will help you keep track of time. Since you will be using this variable in a loop, globalize it. It should look like the following image. 
      global counter
  counter = 0
  3. Since I don't like the default brightness, I lower it 0.01 and to do this I type 
      cpx.pixels.brightness = 0.03 #This step is not required
