Circuit Playground Express

Remote Control Light Switch

By: Rafa Gould and Abby Dhakal

Introduction Outputs Relays

Introduction

In this project, you will work on getting a remote controller and the Circuit Playground Express (CPE) to turn an external light source (such as a lamp) on and off. This project would allow you to manually control certain light fixtures in your room with a click of a button!

What you will make

A python powered remote control switch to any of your favorite gadgets.

What you will learn

  • How to get an output from your Circut Playground Express
  • How relays work and how to operate them
  • How to use remote controls in Circuit Python

What you need

Hardware

  • 1 x Circuit Playground Express
  • 1 x USB A to Micro-B Cable
  • 3 x Alligator Clip to Male Jumper Wires (Red, Yellow, and Black)
  • 3 x Female to Female Jumper Wires
  • 1 x Two Prong Extension Cord
  • 1 x Parallax Single Relay Board
  • 1 x Adafruit IR Remote Control
  • 1 x Roll of Electric Tape
  • 1 x Standard Wire Connector

Software

  • Mu (or any other Python file editor)
  • Adadfruit IRRemote Library
