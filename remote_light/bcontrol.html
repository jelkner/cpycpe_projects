Button Control

We want to produce a minimum viable product before we go all out with the remote let’s start by triggering the relay with a button, this will be really easy. Let’s start up by importing the CPX library so we can use the buttons: 

    from adafruit_circuitplayground.express import cpx

Now let’s add an endless loop, we’re going to do this using: 

    while True:

Now inside of our loop let’s code for turing on the relay using button A. 

    if cpx.button_a:
        relay.value = True

If we press button A the relay will turn on. But how do we turn this off? Using button B to turn the relay off would be an inefficient use of buttons so instead we can use button A as a toggle switch, by adding a condition that the relay.value must be True to turn it off or must be False to turn in on. 

    if cpx.button_a relay.value == False:
        relay.value = True
    elif cpx.button_a and relay.value == True:
        relay.value = False

Now if you plug something into the extension cord and press button A. It should turn on! You’ve completed the challenge, but we promised a remote control light switch, so now we have to work on that!

Final Product

    from adafruit_circuitplayground.express import cpx
    import board
    import digitalio
    
    relay = digitalio.DigitalInOut(board.D12)
    relay.direction = digitalio.Direction.OUTPUT
    relay.value = False
    
    while True:
        if cpx.button_a and relay.value == False:
            relay.value = True
        elif cpx.button_a and relay.value == True:
            relay.value = False
