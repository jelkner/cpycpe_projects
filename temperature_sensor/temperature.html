Temperature

  1. Before you start, be sure to import time and from adafruit_circuitplayground.express import cpx
  2. Create a while loop thats run on forever
  3. Make a variable that stores the temperature value.
  4. Create a if and else statement that checks to see if the temperature exceeds a certain value, if it does then change the color of the Playground Express
  5. It should look something like this in the end : 
            import time
        from adafruit_circuitplayground.express import cpx

        while True:
            temperature_value = cpx.temperature
            cpx.pixels.brightness = 0.01 #Not necessary, I changed it because the default its too bright.
            if temperature_value > 28: #
                cpx.pixels.fill((255,0,0))
            else:
                cpx.pixels.fill((0,0,255))
            time.sleep(0.1)
